3 What We Can Build With Each Data Input

Each data gap unlocks specific analytical capabilities in the Property Intelligence Engine.
Rent + Outgoings

Occupancy Ratio Model

Sales-per-SQM vs. rent-per-SQM efficiency scoring. Identifies over-rented locations and quantifies lease renegotiation opportunity. Feeds directly into the 4-Wall P&L model.

Labour Costs

Labour Productivity Index

Revenue-per-labour-dollar benchmarking across the network. Enables rostering optimisation targeting and highlights stores where headcount does not match throughput.

Fitout Capital

ROI & Payback Calculator

Capital-employed vs. EBITDA payback period per store. Essential for new-site investment cases and refurbishment prioritisation.

EBITDA

True Profitability Ranking

Replaces GPM-based proxy with actual earnings. Powers the Performance x Profitability matrix and validates the CBAS tier assignments.

Store Name Reconciliation

Unified Store Master

Enables automatic joins between POS sales data, CBAS catchment metrics, and ROC competitive landscape. Prerequisite for all cross-dataset analysis.

Ramp-Up Re-Assessment

New Store Trajectory Model

Actual vs. projected performance tracking for 2025 openings. Validates the site-selection model's predictive accuracy and calibrates future projections.

Quantium Penetration

Validated Opportunity Scoring

Upgrades expansion opportunity assessment from 'directional' to 'validated'. Distinguishes true white-space from travel-loyal behaviour in low-share postcodes.

ABS Census Demographics

Demographic Success Profile

Empirically-derived ideal catchment profile (income, household composition, professional/managerial workforce %). Powers automated site-scoring for new locations.
